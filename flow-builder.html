此工具是快速幫大家繪製工作流的 Beta 版，在拉線時難免有繞圈、彎曲的問題，可以自行拖曳各方框之間的距離來調整。
這份工具的核心操作只有五步：先建立單位／角色歸屬、再新增任務與步驟、拖曳圓點畫線、點選連線編輯、最後匯出。以下用簡圖示範。
1
先到左側建立「單位 → 角色」歸屬
開始畫之前，先在左側組織結構按「＋新增單位」（例如你的部門），再在單位下按「＋新增角色」（例如你的職能）。之後每個任務都會歸在某個角色底下，方便篩選、也方便之後多人彙整時自動歸類。
2
在角色下方新增任務、新增步驟／判斷節點
在角色下方按「＋新增任務」，會新增一列橫貫六個階段的 flow；節點滑鼠移上去會浮出頂部小圖示，可以再新增步驟（方形）或判斷（菱形），或刪除。
3
拖曳四邊圓點畫連接線
滑鼠移到節點上會顯示四邊的小圓點，從一個圓點按住拖到另一個節點上放開，就會畫出連接線；線會自動轉直角避開節點，不會用拖曳的話直接穿過去。
4
點一下連接線可以編輯
連接線本身不用 hover，點一下即可選取（再點一次取消選取）。選取後會出現：標籤（可打字，例如 Yes / No / 退回原因）、⟲切換一般線／退回虛線、✕刪除、兩端的圓形把手（拖到別的節點可重新連接）、中間的方形把手（拖曳可調整繞路位置，避免線疊在一起）。
5
用右上角 Export 匯出／匯入
畫好後從右上角「Export ▾」匯出 Markdown 或 JSON；多人分頭畫好後，各自匯出 JSON 交回給你，你用「合併匯入 JSON」逐個彙整進同一份大表，不會覆蓋你已經有的內容。左側任務名稱旁的單位/角色標籤也可以點一下，只看該任務的 flow。
